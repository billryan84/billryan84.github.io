General Outline of Data Analysis

I. Prepare images for doing photometry (link)

  • measure and subtract bias offset (signal added to each image)
  • create master bias and flat frames
  • subtract bias from all images (account for pixel-to-pixel variation in bias)
  • divide all images by flatfield (account for pixel-to-pixel variation in sensitivity of ccd)

    • Object images are now ready to measure magnitudes

II. Perform photometry

  • measure magnitudes of object and comparison stars
  • compute differential magnitudes of object

III. Analysis

  • plot differential magnitude versus time for each night (remember, lower magnitude is brighter); examine plots to estimate period
  • perform Fourier analysis and examine power spectrum to determine potential rotation periods
  • phase back data to see if period 'works'

    • At this point you should have determined the spin rate and the amplitude for the asteroid.
